# Movies Battle API

Api Movies Battle aims to provide contracts through an API for a quiz where the user will have to hit the best rated movie on IMDb.

Base url: https://movies-battle-service.herokuapp.com

 ## Documentation

- Spring Boot
- JUnit 
- Spring Security
